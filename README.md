# CodeAlpha Task 4: Sentiment Analysis

## Description
This project performs sentiment analysis on product reviews using NLP to classify text as Positive, Negative, or Neutral and understand customer emotions.

## Tools Used
Python, Pandas, VADER Sentiment, Matplotlib, Seaborn, WordCloud

## Key Findings
1. Sentiment: 60% Positive, 20% Neutral, 20% Negative
2. 5-star reviews = +0.7 score, 1-star = -0.6 score
3. Top words: love, amazing, quality, best
4. Business use: Use 'amazing quality' in marketing. Fix 'broke' complaints.
